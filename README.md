 **Meesho Clone (Frontend UI)**



A responsive frontend clone of the Meesho e-commerce platform built using pure HTML and CSS.  

This project focuses on recreating the visual layout, structure, and user interface of the Meesho website.



------------------------------------------------------------



  **Features**



\- Clean and modern UI inspired by Meesho

\- Responsive layout for desktop and mobile screens

\- Product listing cards

\- Navigation bar and footer sections

\- Static cart and product detail layouts

\- Reusable CSS variables for easy theming



-------------------------------------------------------------



 **Tech Stack**



\- HTML

\- CSS (Flexbox, Grid, Custom Properties)

\- No JavaScript (UI-only clone)



---------------------------------------------------------------
